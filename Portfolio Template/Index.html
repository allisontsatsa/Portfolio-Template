FizzBuzz

This exercise finds multiples of 3 and replaces them with "Fizz". For multiples of 5, it replaces them with "Buzz" and for numbers that are both, it will return "FizzBuzz".



 $('#btnFizz').on("click", function () {
    var stuff = '';
    for (var i = 1; i < 101; i++) {
    if (i % 3 == 0 && i % 5 == 0)
    stuff += "FizzBuzz, ";
    else if (i % 3 == 0)
    stuff += 'Fizz, ';
    else if (i % 5 == 0)
    stuff += 'Buzz, ';
    else
    stuff += i + ', ';
    }
 $('#output').html(stuff.substr(0, stuff.length - 2));

TacoCat

To determine if something is a palindrome, please enter your text below.



 $('#btnCrunch').click(function () {
            var str = $('#input').val().toLowerCase();
            var reversed = str.split('').reverse().join('');
            if (str === reversed) {
	$('#output').text(str + " This is a palindrome.");
            }
            else {
	$('#output').text("That is not a palindrome.");
            }
        })
